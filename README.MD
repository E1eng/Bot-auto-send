## Langkah-langkah

### 1. Clone Repositori

```bash
git clone https://github.com/E1eng/Bot-auto-send
cd bot-auto-send
```

### 2. Instal Dependensi

```bash
npm install
```

### 3. Konfigurasi .env File

buat file .env di root projek

```bash
SEED_PHRASE="your seed phrase here"
```

### 4. Run script

```bash
npm run start
```

\*make sure nodejs installed <<<<<<
